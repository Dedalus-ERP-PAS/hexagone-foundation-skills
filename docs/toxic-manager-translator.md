# Toxic Manager Translator

## Description

Transforme les messages ou réactions émotionnelles d'un collaborateur en réponses professionnelles et diplomatiques, adaptées aux situations de travail difficiles avec un manager toxique.

## Cas d'utilisation

- Reformuler un message frustré avant de l'envoyer à un supérieur
- Répondre à une demande déraisonnable de manière professionnelle
- Gérer les messages passifs-agressifs ou manipulateurs
- Poser des limites sans créer de conflit
- Documenter les échanges de manière stratégique

## Fonctionnalités

### Framework GRACE
- **G**round in facts - Ancrer dans les faits
- **R**eframe - Recadrer le narratif
- **A**ssert boundaries - Affirmer les limites
- **C**reate documentation - Créer une trace écrite
- **E**xit strategy - Garder les portes ouvertes

### Scénarios couverts
- Délais impossibles
- Vol de crédit
- Messages passifs-agressifs
- Expansion du périmètre
- Critiques publiques
- Gaslighting
- Limites travail/vie personnelle
- Évaluations injustes

## Exemple

**Message original (émotionnel) :**
> "C'est n'importe quoi ! Je ne peux pas finir ça pour vendredi. Tu me donnes toujours des trucs au dernier moment !"

**Message transformé (professionnel) :**
> "Je veux m'assurer qu'on livre un travail de qualité. Pour tenir le délai de vendredi, je devrais déprioritiser [X et Y]. Pourrions-nous discuter des priorités, ou explorer un ajustement du calendrier ?"
